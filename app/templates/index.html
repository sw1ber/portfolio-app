AWS EC2 · Docker · Nginx · FastAPI · GitHub Actions

About This Project

실제 운영 환경을 가정하여 AWS EC2에 웹 서비스를 배포하고, Docker와 GitHub Actions를 이용해 CI/CD 파이프라인을 구축했습니다.

Tech Stack

  • AWS EC2 (Amazon Linux)
  • Docker / Docker Compose
  • Nginx (Reverse Proxy)
  • FastAPI
  • GitHub Actions (CI/CD)

Architecture

Nginx가 80 포트에서 요청을 받아 FastAPI 컨테이너로 전달합니다. 전체 서비스는 Docker Compose로 관리됩니다.

CI/CD Pipeline

  1. GitHub main 브랜치에 push
  2. GitHub Actions 트리거
  3. EC2 서버로 SSH 접속
  4. docker-compose pull & up

Logs & Monitoring

Nginx와 FastAPI 로그를 통해 요청 흐름과 에러를 추적할 수 있습니다.

docker-compose logs nginx
docker-compose logs api

Troubleshooting

  • 80 포트 충돌 → 기존 프로세스 확인 후 컨테이너화
  • StaticFiles 경로 오류 → Docker COPY 경로 수정
  • ModuleNotFoundError → uvicorn 실행 경로 수정
