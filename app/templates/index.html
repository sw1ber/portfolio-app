AWS EC2 · Docker · Nginx · FastAPI · GitHub Actions

About This Project

AWS EC2 환경에서 Docker 기반 웹 서비스를 구축하고 GitHub Actions를 통해 CI/CD 파이프라인을 구성한 개인 포트폴리오입니다.

Tech Stack

  • AWS EC2 (Amazon Linux)
  • Docker / Docker Compose
  • Nginx (Reverse Proxy)
  • FastAPI
  • GitHub Actions

Architecture

Nginx가 80 포트에서 요청을 수신하고 FastAPI 컨테이너로 전달합니다.

CI/CD Pipeline

  1. GitHub main 브랜치에 코드 push
  2. GitHub Actions 자동 실행
  3. EC2 서버 SSH 접속
  4. Docker Compose로 서비스 재배포

Logs & Monitoring

Nginx 및 FastAPI 로그를 통해 요청 흐름과 에러를 추적합니다.

docker-compose logs nginx
docker-compose logs api

Troubleshooting

  • 80 포트 충돌 → 기존 프로세스 종료 후 컨테이너화
  • StaticFiles 경로 오류 → Docker COPY 경로 수정
  • ModuleNotFoundError → uvicorn 실행 경로 수정
